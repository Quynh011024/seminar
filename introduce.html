Bootstrap là gì?

Bootstrap là một framework front-end mã nguồn mở được tạo bởi Mark Otto và Jacob Thornton tại Twitter. Nó ra mắt lần đầu vào tháng 8 năm 2011, với mục tiêu đơn giản hóa việc xây dựng giao diện web hiện đại.

Các cột mốc quan trọng:

  • 2011: Bootstrap 1.0 ra mắt với hệ thống grid cơ bản và các thành phần UI.
  • 2013: Bootstrap 3.0 tập trung vào thiết kế responsive.
  • 2018: Bootstrap 4.0 giới thiệu CSS Flexbox và cải tiến grid system.
  • 2021: Bootstrap 5.0 loại bỏ dependency jQuery, cải thiện performance.
