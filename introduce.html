Bootstrap là gì?

Bootstrap là một framework front-end mã nguồn mở được tạo bởi Mark Otto và Jacob Thornton tại Twitter.

  • Thời gian ra đời: Bootstrap được phát triển vào năm 2010 tại công ty Twitter.
  • Người sáng lập:
    • Mark Otto: Nhà thiết kế giao diện tại Twitter.
    • Jacob Thornton: Kỹ sư phần mềm tại Twitter.
  • Tên gọi ban đầu: Twitter Blueprint, với mục tiêu chuẩn hóa và đơn giản hóa các công cụ giao diện người dùng nội bộ tại Twitter.

  • Vấn đề chính tại Twitter: Trước khi Bootstrap ra đời, đội ngũ phát triển tại Twitter sử dụng nhiều công cụ khác nhau, gây nên sự không đồng nhất trong giao diện và trải nghiệm người dùng.
  • Giải pháp: Bootstrap được tạo ra như một framework thống nhất, kết hợp giữa HTML, CSS, và JavaScript để xây dựng giao diện nhất quán, dễ bảo trì.

  • Ngày công bố: Bootstrap chính thức được phát hành dưới dạng mã nguồn mở trên GitHub vào ngày 19/8/2011.
  • Mục tiêu công khai: Mang lại một framework front-end dễ sử dụng, giúp cộng đồng lập trình viên xây dựng giao diện nhanh chóng và hiệu quả.

Bootstrap nhanh chóng trở thành một trong những framework phổ biến nhất trên GitHub, được sử dụng rộng rãi từ các trang web cá nhân đến các hệ thống phức tạp.

Cột Mốc Quan Trọng

2011: Bootstrap 1.0 ra mắt
2012: Bootstrap 2.0 tập trung responsive
2013: Bootstrap 3.0 phát hành
2018: Bootstrap 4.0 Flexbox
2021: Bootstrap 5.0 loại bỏ jQuery

Các Phiên Bản Bootstrap

Bootstrap 1 (2011)

Ngày phát hành: 19/8/2011

  • Hệ thống lưới cơ bản
  • Các thành phần giao diện HTML và CSS
  • Tích hợp Glyphicons

Hạn chế: Hỗ trợ hạn chế cho thiết bị di động.

Bootstrap 2 (2012)

Ngày phát hành: 31/1/2012

  • Thiết kế Responsive Design
  • Thêm các thành phần UI mới như navbar

Điểm đặc biệt: Hỗ trợ tốt hơn cho thiết bị di động.

Bootstrap 3 (2013)

Ngày phát hành: 19/8/2013

  • Mobile-first design
  • Cải tiến lưới với breakpoints mới

Nhược điểm: Nâng cấp khó khăn từ Bootstrap 2.

Bootstrap 4 (2018)

Ngày phát hành: 18/1/2018

  • Tích hợp Flexbox
  • Hỗ trợ Sass

Nhược điểm: File CSS và JS lớn.

Bootstrap 5 (2021)

Ngày phát hành: 5/5/2021

  • Loại bỏ jQuery
  • Bổ sung hỗ trợ CSS Grid

Điểm mạnh: Tập trung tối ưu trình duyệt hiện đại.
